Java

Java teknolojisi nedir ve neden ihtiyacım var?

Java, 1995 yılında Sun Microsystems tarafından yayınlanmış bir hesaplama platformu ve programlama dilidir. Basit başlangıçlar ile ortaya çıkmıştır ve birçok hizmet ve uygulamanın oluşturulduğu güvenilir platformu sağladığı için bugünün dijital dünyasında en büyük paya sahiptir. Gelecekte kullanmak üzere tasarlanmış yeni, inovatif ürünler ve dijital hizmetler de Java'yı temel almaya devam ediyor.

  1. java.com üzerinden Java yazılımını indirdiğimde ne alacağım?
    • Java yazılımını java.com üzerinden indirdiğinizde Java Çalıştırma Zamanı Ortamı (JRE) sürüm 8 elde edersiniz.
  2. Java Eklenti yazılımı nedir?
    • Java Eklentisi yazılımı, Java Çalıştırma Zamanı Ortamı'nın (JRE) bir bileşenidir. Java Çalıştırma Zamanı Ortamı (JRE), Java programlama dilinde yazılan bazı uygulamaların bazı tarayıcılar yoluyla başlatılmasına izin verir. Java Eklentisi yazılımı bağımsız bir program değildir ve ayrı olarak yüklenemez.
  3. Java Sanal Makinesi ve JVM hükümlerini biliyorum. Bu Java yazılımı mı?
    • Java Sanal Makinesi bir uygulamanın çalıştırılmasına dahil olan Java yazılımının sadece bir parçasıdır. Java Sanal Makinesi JRE'nin bir parçası olarak doğrudan Java yazılımı indirmenizin içinde oluşturuldu ve Java uygulamalarının çalıştırılmalarına yardımcı oluyor.
